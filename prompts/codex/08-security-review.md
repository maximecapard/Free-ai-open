# Codex task — security review

## Objective

Audit the repository for security/privacy issues.

## Focus areas

- XSS risks.
- CSP gaps.
- Telemetry leaks.
- Forbidden fields.
- Insecure model URLs.
- Missing model license/hash metadata.
- Browser-only API misuse.
- Overly precise fingerprinting.
- Server endpoints accepting arbitrary JSON.
- Dependency risks.

## Output format

- Critical issues
- High priority
- Medium priority
- Low priority
- Recommended tests
- Recommended docs changes
