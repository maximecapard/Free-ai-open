# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for FreeAI Open.

ADRs are short documents that record decisions with long-term architectural or privacy impact. They should explain the context, the decision, consequences, and any known limits.

## Index

- [ADR 0001 - Keep inference local by default](0001-local-first-browser-inference.md)
- [ADR 0002 - Store only technical local logs and diagnostic reports](0002-technical-local-diagnostics.md)

## Template

```md
# ADR NNNN - Title

## Status

Accepted | Proposed | Superseded

## Context

What problem or constraint led to this decision?

## Decision

What are we deciding?

## Consequences

What gets easier, harder, or explicitly out of scope?
```
